Git

  • git status: checks what branch we are currently on
  • git checkout (): switch to another branch and check it out in youyr working directory
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: adds all modifications in current working branch to staging area
  • git commit m-: commits to repository, m- associates mesage with our commit
  • git pull origin main: ideal to pull base branch into feature before pushing up code (don't want to be working on an older file)
  • git push origin folder/place: pushes changes
